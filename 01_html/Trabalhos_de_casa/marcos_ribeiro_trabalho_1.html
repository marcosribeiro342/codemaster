Perguntas sobre HTML

Marcos Ribeiro

1 - O que é HTML?

HTML é uma linguagem de marcação criada por Tim Berners-lee, é utilizada para estruturar e defenir o conteúdo de uma página web. A linguagem de marcação é utilizada para anotar conteúdos como textos, imagens, tabelas e listas para posterior exibição no browser.

2 - O que é um elemento HTML? dê exemplos.

Um documento HTML é constituído por um conjunto de elementos dispostos de forma hierarquizada. Os elementos são constituídos por tags que definem qual o tipo de conteúdo por exemplo a tag <p> serve para representar um parágrafo, um titulo grande sera representado pela tag <h1>, Apenas o conteúdo apresentado no elemento será apresentado no seu navegador, as tags apenas apareceram no deu documento HTML.

3 - Que ferramentas são necessárias para construir um site HTML?

Para a construção de um site HTML é necessário apenas um editor de texto, como o bloco de notas, mas o mais usual é usar um programa especifico como o virtual studio code, entre outros.

4 - Quais os elementos base necessários para a criação de uma página HTML?

Os elementos base para fazer uma página HTML são:
<!DOCTYPE html>
<html>
<head>
</head>
<body>
</body>
</html>

5 - O elemento "br" necessita ser fechado?

Como é um elemento vazio não precisa ser fechado.

6 - Para que servem os atributos?

Os atributos são informações que passamos na tag para que ela se comporte da maneira esperada. Existem alguns atributos globais pois funcionam em todas as tags e especificos que são utilizados para cada tag.
